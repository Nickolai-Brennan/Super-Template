# Architecture Decision Records (ADRs)

Use ADRs to capture meaningful technical and process decisions.

## How to add an ADR
1. Copy [`/templates/adr-template.md`](../../templates/adr-template.md).
2. Name the file `adr-XXXX-short-title.md` (zero-padded sequence).
3. Set status and context clearly.
4. Link related docs, PRs, and issues.
5. Keep old ADRs immutable; create a new ADR to supersede.
