# Runbooks

Runbooks document repeatable operational procedures for support, incidents, and maintenance.

## How to add a runbook
1. Copy [`/templates/runbook-template.md`](../../templates/runbook-template.md).
2. Name the file with a clear operation scope (for example `incident-response.md`).
3. Define trigger conditions, steps, and rollback.
4. Add verification checks and escalation contacts.
